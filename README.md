# snowwolf-script
集成dns，whois，扫描端口，shodan，sqlmap辅助工具（后期决定删除改为帮助指南），自动化exploit工具
# 安装
apt-get install shodan

pip install -r requirements.txt

chmod +x Snowwolf.sh

请自行安装python2.7环境

安装完成后运行即可
